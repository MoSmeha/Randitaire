# Solitaire Leaderbaord App

## Overview

This web app lets users enter their name, generate a random score for Solitaire, and submit it to a leaderboard.  
The app displays all submitted players and their scores, making it easy to track and compare results. It also shows link to the real solitaire app.

---

## Prerequisites

Before running the app, make sure you have:

- **XAMPP** installed
- **PHP 8.2+, MySQL (included with XAMPP)**

---

## Setup Instructions

1. **Install and Start XAMPP**

   - Open the XAMPP Control Panel.
   - Start **Apache** and **MySQL**.

2. **Move the Project Folder**

   - Place this project folder inside htdocs folder inside xampp folder

3. **Create the Database**

   - Open your browser and go to:
     ```
     http://localhost/phpmyadmin
     ```
   - Create a new database, for example:
     ```
     solitaire_db
     ```

4. **Run the App**
   - Open your browser and visit:
     ```
     http://localhost/solitaire-app/
     ```

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP (via XAMPP)
- **Database:** MySQL
